# RNA-Seq Analysis Projects in R

A collection of transcriptomics and RNA-Seq analysis projects implemented in R and Bioconductor.

## Projects

### 1. Cardiomyopathy Transcriptomic Analysis
- GEO data integration
- Batch correction using ComBat
- Differential expression analysis using Limma
- PCA, Heatmaps, Volcano plots

### 2. Time-Course RNA-Seq Analysis using DESeq2 LRT
- Likelihood Ratio Test (LRT)
- Temporal gene expression analysis
- Gene clustering
- Time-course visualization

## Tools
- R
- DESeq2
- Limma
- GEOquery
- ggplot2
- pheatmap
- Quarto
