This folder include all markdown , html files of Deciphering Cardiomyopathy Through GEO Microarray Analysis with R

**Project description:**

1)Integrated six public GEO microarray datasets to investigate the role of Z-disc-associated genes in ischemic and non-ischemic cardiomyopathy. 
2)Performed data preprocessing, probe annotation, batch-effect correction using ComBat, differential expression analysis with limma, PCA, heatmaps, volcano plots, and statistical validation. 
3)Identified key dysregulated genes including FLNC, FHOD3, MYOZ2, CRYAB, PDLIM3, and LDB3, providing insights into cardiomyopathy-related molecular mechanisms.
